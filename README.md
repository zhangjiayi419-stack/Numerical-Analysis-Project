# Numerical-Analysis-Project
Python implementation of numerical analysis algorithms including interpolation, integration, root-finding, and ODE solvers
This repository contains the source code and the comprehensive project report for the MATH2007: Numerical Analysis course at the University of Macau. The project focuses on the theoretical exploration and Python-based implementation of core numerical methods.
This project provides an in-depth analysis and implementation of four fundamental topics in numerical analysis:
Interpolation Algorithms: Comparison of Lagrange, Newton, and Cubic Spline interpolation methods to analyze the Runge phenomenon and the stability of equidistant nodes.
Numerical Integration: Construction and evaluation of two-point quadrature formulas, including specialized Gauss-Legendre and weighted Jacobi integration.
Root-finding Methods: Evaluation of convergence efficiency and reliability for the Bisection method, Newton's method, and the Secant method.
Numerical Solutions for ODEs: Investigation of stiffness in differential equations by comparing the stability and accuracy of explicit methods (e.g., Modified Euler) versus A-stable implicit methods (e.g., Trapezoidal rule).
